This is the React (Vite) frontend for the Political Promise Tracker. For full project documentation, setup, and API details, see the root `README.md` in the repository.

Quick links:
- Frontend dev: `npm run dev`
- Build: `npm run build`
- Preview: `npm run preview`

Environment:
- `VITE_API_URL` should point to the running API, e.g. `http://localhost:5000`.

Features:
- Dashboard with completion charts, status distribution, leaderboard, and tracker
- Ministers list and profiles with monthly trends and promises
- Promise list with status filtering and related news
- Admin page for login and data refresh/import actions
